When the "Start" button is clicked, the startRoll function first checks whether the rolling variable is false. If it is, it sets the rolling variable to true and starts the dice rolling by setting the intervalId and calling the rollDice function repeatedly. If the rolling variable is already true, the startRoll function does nothing.
When the "Stop" button is clicked, the stopRoll function first checks whether the rolling variable is true. If it is, it stops the dice from rolling by clearing the intervalId. It then gets the rolled number by parsing the text content of the dice element, and uses a switch statement to show the appropriate alert. Finally, it sets the rolling variable back to false.
With this modification, the alert is shown after the "Stop" button is clicked and the dice has stopped rolling, and the dice won't start rolling again until the "Start" button is clicked.
