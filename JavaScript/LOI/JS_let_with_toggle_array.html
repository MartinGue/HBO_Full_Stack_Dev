JavaScript const

Click the button to toggle between two constant arrays:

This code defines two different constant arrays (cars1 and cars2),
and then initializes a variable called "cars" to the first array.
When the button is clicked, the toggleCars() function is called,
which checks the current value of the "cars" variable
and swaps it with the other array accordingly.
The new value of "cars" is then displayed on the page.
