Enter a temperature in Fahrenheit:

This code adds an input box with the ID "fahrenheitInput", and sets an "oninput" attribute to call a new function called "updateCelsius()" whenever the input value changes. The "updateCelsius()" function gets the current value of the input box, calls the "toCelsius()" function to convert the value to Celsius, and then updates the content of the "demo" paragraph with the new result.
